# Figure index — maps each file to its slide

All figures generated by `analysis/make_slides.py` (regenerate with
`python3 analysis/make_slides.py` from the repo root). Filenames are slide-numbered so
they bind to `DECK_PROMPT.md`. Color convention is constant:
**blue = glass, gold = frame, green = ground, red = interior.** All EDA figures use REAL
labelled scans (no synthetic stand-ins).
